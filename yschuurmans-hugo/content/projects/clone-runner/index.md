Clonerunner was my first project I aimed to finish from start to end, since this was the first time I got the opportunity to develop something from scratch, without a set goal. This gave me the motivation to work long days and even longer nights to create something I would like to play.

However, the longer I worked on the project, the harder it became to add new content. I started to realise the mistakes I had made, the wrong decisions that would prevent me from adding certain features unless I rewrote that piece of code. After many attempts I decided to drop the project, promising myself to try to use what I learned as a stepping stone for better projects.
